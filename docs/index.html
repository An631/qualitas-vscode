Getting Started

Choose your path: use the VS Code extension for real-time feedback, or the npm package for CLI and CI pipelines

1

Download the extension

Grab the latest .vsix from GitHub Releases and install it.

code --install-extension qualitas-vscode-x.x.x.vsix
2

Open a project

Open any TypeScript, JavaScript, Python, or Rust project in VS Code. Qualitas activates automatically and begins analyzing your files.

3

See scores instantly

Every function and class gets a CodeLens score above it (e.g. Score A: 92.3). Functions with issues get squiggly underlines on the name.

4

Fix with AI

Click the lightbulb on any squiggled function and choose "Fix with AI" to send the issue to your AI assistant (Copilot, etc.) with full context.

5

Configure your rules

Add a qualitas.config.js to your project root to set the profile, thresholds, and per-language flags. The whole team shares the same rules.

module.exports = { profile: "strict", threshold: 80 };
1

Install the package

Add qualitas as a dev dependency to your project.

npm install --save-dev qualitas
2

Analyze from the CLI

Run qualitas on your project to see a full quality report with scores for every function.

npx qualitas ./ -f detail
3

Add to your scripts

Add quality checks to your package.json for easy access and CI integration.

"scripts": { "quality": "qualitas ./ -f detail" }
4

Use the API programmatically

Import qualitas in your code for custom analysis, CI gates, or automation pipelines.

const { analyzeSource } = require("qualitas");
const report = analyzeSource(source, "file.ts");
console.log(report.score, report.grade);
5

Configure

Create a qualitas.config.js in your project root to customize profiles, weights, flag thresholds, and per-language settings.

module.exports = {
  profile: "strict",
  threshold: 80,
  exclude: ["vendor", "generated"],
};

Easy Configuration

Add a qualitas.config.js to your project root and the whole team shares the same rules

// qualitas.config.js
module.exports = {
  profile: "strict",
  threshold: 80,
  flags: {
    tooManyParams: { warn: 3, error: 5 },
    excessiveReturns: false,
  },
  languages: {
    typescript: {
      flags: { deepNesting: { warn: 3, error: 5 } }
    }
  }
};
